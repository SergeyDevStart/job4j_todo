# job4j_todo

## Описание проекта
job4j_todo — это учебный проект, который представляет собой просто список дел.
Пользователь может создавать/ редактировать задачи. Все записи сохраняются в базу данных.  

## Используемые технологии в проекте:
- Spring boot
- Thymeleaf
- Bootstrap
- PostgreSQL
- hibernate
- Liquibase
